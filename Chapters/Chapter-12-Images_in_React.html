Required Attributes

Optional Attributes

The React Way: Import as Module

When you import an image in React, the build tool (Vite/Webpack) processes it, optimizes it, and gives you a URL that includes a content hash for cache-busting.

üì¶ Basic Import Syntax

Using the Public Folder

Images in the public folder are served as-is, without processing. They're accessible via absolute paths from the root URL.

üìÇ Public Folder Structure

üìÅ my-react-app/
üìÅ public/
üìÑ index.html
üñºÔ∏è favicon.ico

üñºÔ∏è hero.jpg
üìÅ src/
üìÑ App.jsx

‚ö†Ô∏è When to Use Public Folder

  • Dynamic image paths (user-selected)
  • Many images (100+)
  • Images referenced in HTML directly
  • Files that must keep exact names
  • Large files you don't want bundled

üìù Using Public Folder Images

‚öñÔ∏è Import vs Public Folder

Aspect Import (src/assets) Public Folder

üí° Rule of Thumb
Default: Import images (better optimization)

Loading Images Dynamically

Sometimes you need to load different images based on props, state, or user input. Here are several approaches to handle dynamic image loading.

üéØ Method 1: Object Map (Best for Known Images)

  ImageMap Approach
// Import all possible images
import gradeA from './assets/grades/grade-a.png';
import gradeB from './assets/grades/grade-b.png';
import gradeC from './assets/grades/grade-c.png';
import gradeD from './assets/grades/grade-d.png';
import gradeF from './assets/grades/grade-f.png';

// Create a map
const gradeImages = {
  'A': gradeA,
  'B': gradeB,
  'C': gradeC,
  'D': gradeD,
  'F': gradeF,
};

function GradeBadge({ grade }) {
  return (
    <img 
      src={gradeImages[grade]}
      alt={`Grade ${grade}`}
      className="grade-badge"
    />
  );
}

// Usage
<GradeBadge grade="A" />
<GradeBadge grade="B" />

üéØ Method 2: Vite's import.meta.glob (Vite Projects)

  Vite Glob Import
// Import all images from a folder at once!

  eager: true,
  query: '?url',
  import: 'default'

// images is an object like:
// {
//   './assets/products/product-1.jpg': '/src/assets/products/product-1.jpg',
//   './assets/products/product-2.jpg': '/src/assets/products/product-2.jpg',
// }

function ProductImage({ productId }) {

    <img 
      src={imageUrl}
      alt={`Product ${productId}`}
    />
  );

üéØ Method 3: Public Folder (Dynamic Paths)

  Public Folder Dynamic
// For truly dynamic images (e.g., from database, API)

function StudentAvatar({ studentId }) {

  return (
    <img 
      src={`/avatars/student-${studentId}.jpg`}
      alt="Student avatar"

}

// API-provided image URLs

    <img 
      src={user.avatarUrl || '/default-avatar.png'}
      alt={user.name}
    />
  );
}

üéØ Method 4: Dynamic Import (Code Splitting)

  Lazy Image Loading
import { useState, useEffect } from 'react';

function DynamicImage({ imageName }) {
  const [imageSrc, setImageSrc] = useState(null);

  useEffect(() => {
    // Dynamically import the image
    import(`./assets/${imageName}.jpg`)
      .then((module) => {
        setImageSrc(module.default);
      })
      .catch((err) => {
        console.error('Failed to load image', err);
      });
  }, [imageName]);

  if (!imageSrc) return <div>Loading...</div>;

  return <img src={imageSrc} alt={imageName} />;
}
üìù Which Method to Choose?
Known set of images: Object Map (Method 1)
Folder of images (Vite): import.meta.glob (Method 2)
User uploads / API images: Public folder or URLs (Method 3)
Code-split large images: Dynamic import (Method 4)

Part 7: Image Optimization
React
‚ñº

Making Images Load Fast

Images are often the heaviest assets on a webpage. Proper optimization can dramatically improve loading times and user experience.

üìè Right Size for the Job

Image Sizing Rules

  • Don't serve 4000px images for 400px displays
  • Resize images to max dimensions they'll be shown at
  • For retina: 2x the display size (e.g., 800px for 400px display)
  • Consider different sizes for different breakpoints

üì∏ Choose the Right Format

JPEG/JPG

  • Photos, complex images
  • No transparency
  • Good compression

PNG

  • Transparency needed
  • Simple graphics
  • Larger file size

WebP ‚≠ê

  • Best of both worlds
  • 30% smaller than JPEG
  • Supports transparency

üéØ Optimized Image Component

  OptimizedImage.jsx
function OptimizedImage({ 
  src, 
  alt, 
  width, 
  height,
  className = '',
  priority = false 
}) {
  return (
    <img
      src={src}
      alt={alt}
      width={width}
      height={height}
      className={className}
      {/* Lazy load non-priority images */}
      loading={priority ? 'eager' : 'lazy'}
      {/* Decode async for smoother loading */}
      decoding="async"
      {/* Prevent layout shift */}
      style={{
        aspectRatio: width && height ? `${width} / ${height}` : 'auto',
        objectFit: 'cover',
      }}
    />
  );
}

// Usage
<OptimizedImage
  src="/images/hero.webp"
  alt="Hero banner"
  width={1200height={600}
  priority  {/* Above the fold - load immediately */}
/>

function PictureWithFallback({ src, alt }) {
  // Remove extension from src
  const basePath = src.replace(/\.[^.]+$/, '');
  
  return (

  LazyImage.jsx
import { useState } from 'react';

function LazyImage({ src, alt, width, height, placeholder }) {
  const [isLoaded, setIsLoaded] = useState(false);
  const [hasError, setHasError] = useState(false);

  return (
    <div
      style={{
        position: 'relative',
        width,
        height,
        overflow: 'hidden',
        backgroundColor: '#1f2937',
        borderRadius: '12px',
      }}
    >
      {/* Placeholder/Loading state */}
      {!isLoaded && !hasError && (
        <div
          style={{
            position: 'absolute',
            inset: 0,
            display: 'flex',
            alignItems: 'center',
            justifyContent: 'center',
            background: placeholder || 'linear-gradient(135deg, #1f2937, #374151)',
          }}
        >
          <span style={{ color: '#9ca3af' }}>Loading...</span>
        </div>
      )}

      {/* Actual image */}
      {!hasError && (
        <img
          src={src}
          alt={alt}
          loading="lazy"
          onLoad={() => setIsLoaded(true)}
          onError={() => setHasError(true)}
          style={{
            width: '100%',
            height: '100%',
            objectFit: 'cover',
            opacity: isLoaded ? 1 : 0,
            transition: 'opacity 0.3s ease',
          }}
        />
      )}

      {/* Error state */}
      {hasError && (
        <div style={{ color: '#ef4444', textAlign: 'center' }}>
          Failed to load image
        </div>
      )}
    </div>
  );
}

Handling Image Load Failures

Images fail to load for many reasons: network issues, wrong URLs, deleted files. Always have a fallback plan!

üéØ Simple Fallback with onError

  Simple Fallback
import defaultAvatar from './assets/default-avatar.png';

function UserAvatar({ user }) {
  return (
    <img
      src={user.avatarUrl}
      alt={user.name}
      onError={(e) => {
        e.target.src = defaultAvatar;  // Replace with fallback
        e.target.onerror = null;      // Prevent infinite loop
      }}
    />
  );
}

üéØ Robust Image Component

  ImageWithFallback.jsx
import { useState } from 'react';

function ImageWithFallback({
  src,
  fallbackSrc,
  alt,
  fallbackElement,
  className,
  ...props
}) {
  const [imgSrc, setImgSrc] = useState(src);
  const [hasError, setHasError] = useState(false);

  const handleError = () => {
    if (!hasError) {
      setHasError(true);
      if (fallbackSrc) {
        setImgSrc(fallbackSrc);
      }
    }
  };

  // If both original and fallback failed, show fallback element
  if (hasError && !fallbackSrc) {
    return fallbackElement || (
      <div
        className={className}
        style={{
          display: 'flex',
          alignItems: 'center',
          justifyContent: 'center',
          background: '#374151',
          color: '#9ca3af',
        }}
      >
        üì∑
      </div>
    );
  }

  return (
    <img
      src={imgSrc}
      alt={alt}
      className={className}
      onError={handleError}
      {...props}
    />
  );
}

// Usage
<ImageWithFallback
  src={user.photo}
  fallbackSrc="/default-user.png"
  alt={user.name}
  fallbackElement={<UserIcon />}

üéØ Avatar Component with Initials Fallback

  Avatar.jsx
import { useState } from 'react';

function Avatar({ src, name, size = 48 }) {
  const [showImage, setShowImage] = useState(!!src);
  
  // Get initials from name
  const initials = name
    .split(' ')
    .map(word => word[0])
    .join('')
    .toUpperCase()
    .slice(0, 2);
  
  // Generate color from name
  const colors = ['#ef4444', '#f59e0b', '#22c55e', '#3b82f6', '#8b5cf6'];
  const colorIndex = name.charCodeAt(0) % colors.length;
  const bgColor = colors[colorIndex];

  const containerStyle = {
    width: size,
    height: size,
    borderRadius: '50%',
    overflow: 'hidden',
    backgroundColor: bgColor,
    display: 'flex',
    alignItems: 'center',
    justifyContent: 'center',
    color: 'white',
    fontWeight: '600',
    fontSize: size * 0.4,
  };

  return (
    <div style={containerStyle}>
      {showImage ? (
        <img
          src={src}
          alt={name}
          style={{ width: '100%', height: '100%', objectFit: 'cover' }}
          onError={() => setShowImage(false)}
        />
      ) : (
        <span>{initials}</span>
      )}
    </div>
  );
}

  Avatar.jsx
import { useState } from 'react';
import defaultAvatar from '../assets/default-avatar.png';
import styles from './Avatar.module.css';

function Avatar({ src, name, size = 'md' }) {
  const [imageSrc, setImageSrc] = useState(src || defaultAvatar);
  const [isLoaded, setIsLoaded] = useState(false);

  const handleError = () => {
    setImageSrc(defaultAvatar);
  };

  return (
    <div className={`${styles.avatar} ${styles[size]}`}>
      {!isLoaded && <div className={styles.skeleton} />}
      <img
        src={imageSrc}
        alt={name}
        loading="lazy"
        onLoad={() => setIsLoaded(true)}
        onError={handleError}
        style={{ opacity: isLoaded ? 1 : 0 }}
      />
    </div>
  );
}

export default Avatar;
  GradeBadge.jsx
// Import grade images
import gradeA from '../assets/grades/grade-a.svg';
import gradeB from '../assets/grades/grade-b.svg';
import gradeC from '../assets/grades/grade-c.svg';

const gradeImages = {
  'A': gradeA,
  'B': gradeB,
  'C': gradeC,
};

const gradeColors = {
  'A': '#22c55e',
  'B': '#3b82f6',
  'C': '#f59e0b',
};

function GradeBadge({ grade }) {
  const image = gradeImages[grade];
  const color = gradeColors[grade] || '#6b7280';

  if (image) {
    return <img src={image} alt={`Grade ${grade}`} width="32" height="32" />;
  }

  // Fallback for unknown grades
  return (
    <span
      style={{
        display: 'inline-flex',
        alignItems: 'center',
        justifyContent: 'center',
        width: 32,
        height: 32,
        borderRadius: '50%',
        backgroundColor: color,
        color: 'white',
        fontWeight: 'bold',
      }}
    >
      {grade}
    </span>
  );
}

export default GradeBadge;
  StudentCard.jsx
import { useState } from 'react';
import { Edit, Trash2, Check } from 'lucide-react';
import Avatar from './Avatar';
import GradeBadge from './GradeBadge';
import styles from './StudentCard.module.css';

function StudentCard({ student, onSelect, onDelete, isSelected }) {
  return (
    <div className={`${styles.card} ${isSelected ? styles.selected : ''}`}>
      {/* Header with Avatar and Grade */}
      <div className={styles.header}>
        <Avatar
          src={student.avatarUrl}
          name={student.name}
          size="lg"
        />
        <GradeBadge grade={student.grade} />
      </div>

      {/* Student Info */}
      <div className={styles.info}>
        <h3 className={styles.name}>{student.name}</h3>
        <p className={styles.details}>
          Age: {student.age} | Class: {student.class}
        </p>
      </div>

      {/* Actions */}
      <div className={styles.actions}>
        <button
          onClick={() => onSelect(student.id)}
          className={`${styles.btn} ${isSelected ? styles.btnSelected : styles.btnPrimary}`}
        >
          {isSelected ? (
            <><Check size={16} /> Selected</>
          ) : (
            'Select'
          )}
        </button>
        <button
          onClick={() => onDelete(student.id)}
          className={`${styles.btn} ${styles.btnDanger}`}
        >
          <Trash2 size={16} />
        </button>
      </div>
    </div>
  );
}

export default StudentCard;

üéâ Summary: Image Handling in React

Technique When to Use
Import Static, known images (recommended default)
Public folder Dynamic paths, many images, API URLs
SVG as component Icons that need styling/animation
Icon library UI icons (Lucide, Heroicons)
Lazy loading Below-the-fold images
Error fallback User avatars, external images
üí° Key Takeaways
1. Always include alt text for accessibility
2. Specify dimensions to prevent layout shift
3. Use lazy loading for images below the fold
4. Handle errors gracefully with fallbacks
5. Optimize images before adding to project
6. Use WebP format when possible
