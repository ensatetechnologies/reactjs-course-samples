What is the DOM?
‚ñº

The Document Object Model (DOM) is a programming interface for web documents. It represents the page as a tree of objects that JavaScript can interact with.

üîë Key Concepts

  • Document - The entire HTML page
  • Object - Each element becomes an object
  • Model - Structured representation (tree)

‚ú® What Can You Do?

  • Change HTML content & attributes
  • Modify CSS styles
  • Add/remove elements
  • React to user events
üí° Think of it like this...
If HTML is a blueprint of a house, the DOM is the actual house built from that blueprint. JavaScript is like a contractor who can modify the house after it's built!

3
Selecting DOM Elements
‚ñº

Before you can manipulate elements, you need to select them. Here are the main methods:

getElementById()
Select ONE element by its ID (fastest)
const title = document.getElementById('title')
// Returns: single element or null
getElementsByClassName()
Select ALL elements with a class
const buttons = document.getElementsByClassName('btn')
// Returns: HTMLCollection (array-like)
querySelector()
Select FIRST element matching CSS selector
const firstCard = document.querySelector('.card')
const header = document.querySelector('header nav')
// Returns: single element or null
querySelectorAll()
Select ALL elements matching CSS selector
const allCards = document.querySelectorAll('.card')
const links = document.querySelectorAll('a[href^="http"]')
// Returns: NodeList (can use forEach)

4
Manipulating DOM Elements
‚ñº

Once you've selected an element, you can modify it in many ways:

üìù Changing Content

  content.js
const element = document.getElementById('myElement')

// Change text content (safe, no HTML parsing)
element.textContent = 'New text content'

// Change HTML content (can include tags)
element.innerHTML = '<strong>Bold</strong> text'

// Get/Set attributes
element.getAttribute('href')
element.setAttribute('href', 'https://example.com')

// Shortcut for common attributes
element.id = 'newId'
element.className = 'new-class'
element.src = 'image.png'

üé® Changing Styles

  styles.js
const box = document.querySelector('.box')

// Direct style manipulation (inline styles)
box.style.backgroundColor = '#22d3ee'
box.style.padding = '20px'
box.style.borderRadius = '10px'  // Note: camelCase!

// Better: Toggle CSS classes
box.classList.add('active')
box.classList.remove('hidden')
box.classList.toggle('dark-mode')
box.classList.contains('active')  // true/false

// Replace class
box.classList.replace('old-class', 'new-class')

5
Creating & Removing Elements
‚ñº

You can dynamically add or remove elements from the DOM:

‚ûï Creating Elements

// 1. Create the element
const newDiv = document.createElement('div')

// 2. Add content and attributes
newDiv.textContent = 'I am new!'
newDiv.className = 'card new-card'
newDiv.id = 'card-1'

// 3. Add to the DOM
document.body.appendChild(newDiv)

// Or insert at specific position
parent.insertBefore(newDiv, referenceElement)

// Modern methods
parent.append(newDiv)        // At end
parent.prepend(newDiv)       // At start
element.before(newDiv)       // Before element
element.after(newDiv)        // After element

‚ûñ Removing Elements

// Modern way (recommended)
element.remove()

// Old way (still works)
parent.removeChild(element)

// Remove all children
parent.innerHTML = ''

// Or more safely
while (parent.firstChild) {
  parent.removeChild(parent.firstChild)
}

// Replace an element
oldElement.replaceWith(newElement)

üéß Adding Event Listeners

  events.js
const button = document.querySelector('#myButton')

// Method 1: addEventListener (recommended)
button.addEventListener('click', function(event) {
  console.log('Button clicked!')
  console.log(event.target)  // The clicked element
})

// Method 2: Arrow function
button.addEventListener('click', (e) => {
  console.log('Clicked!', e)
})

// Method 3: Named function (can be removed later)
function handleClick(e) {
  console.log('Handling click')
}
button.addEventListener('click', handleClick)
button.removeEventListener('click', handleClick)

// Common events
element.addEventListener('click', fn)      // Mouse click
element.addEventListener('dblclick', fn)   // Double click
element.addEventListener('mouseenter', fn) // Mouse enters
element.addEventListener('mouseleave', fn) // Mouse leaves
element.addEventListener('keydown', fn)    // Key pressed
element.addEventListener('keyup', fn)      // Key released
element.addEventListener('input', fn)      // Input value changes
element.addEventListener('submit', fn)     // Form submitted
element.addEventListener('focus', fn)      // Element focused
element.addEventListener('blur', fn)       // Element loses focus

Traversing the DOM Tree
‚ñº

Navigate between related elements in the DOM tree:

Click any node to see its relationships:

selected element

.firstElementChild
<child>

  traversal.js
const element = document.querySelector('.current')

// Parent
element.parentElement          // Direct parent
element.closest('.container')   // Nearest ancestor matching selector

// Children
element.children               // All child elements (HTMLCollection)
element.firstElementChild      // First child element
element.lastElementChild       // Last child element
element.childElementCount      // Number of children

// Siblings
element.previousElementSibling // Previous sibling element
element.nextElementSibling     // Next sibling element

// Example: Highlight all siblings
let sibling = element.parentElement.firstElementChild
while (sibling) {
  if (sibling !== element) {
    sibling.classList.add('highlight')
  }
  sibling = sibling.nextElementSibling
}
üí° Element vs Node Properties
Use Element versions (e.g., parentElement, children) to skip text nodes and comments. Node versions (parentNode, childNodes) include all node types.
