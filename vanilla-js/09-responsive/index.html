üí° Best Practices:
  • Use rem for typography and spacing
  • Use % or fr for layout widths
  • Use vh/vw for full-screen sections
  • Avoid px for layout (OK for borders, shadows)

.hero-image {

  height: 300px;
  object-fit: cover;
  object-position: center;
}

üîß Modern CSS: clamp()

Responsive values without media queries

This heading uses clamp()

The clamp(min, preferred, max) function creates responsive values that smoothly scale between a minimum and maximum.

clamp.css 
/* Responsive font-size with clamp */
.heading {
  font-size: clamp(1.5rem, 4vw, 3rem);
  /* min: 1.5rem, preferred: 4vw, max: 3rem */
}

/* Responsive container width */
.container {
  width: min(100% - 2rem, 1200px);
  margin-inline: auto;
}

/* Responsive padding */
.section {
  padding: clamp(1rem, 5vw, 4rem);
}
