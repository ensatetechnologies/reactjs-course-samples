Selection Methods

getElementById()

Selects ONE element by its ID (fastest method)

document.getElementById('unique-element')

getElementsByClassName()

Selects ALL elements with a class (returns HTMLCollection)

document.getElementsByClassName('demo-item')

querySelector()

Selects FIRST element matching CSS selector

document.querySelector('.demo-item')

querySelectorAll()

Selects ALL elements matching CSS selector (returns NodeList)

document.querySelectorAll('.demo-item')

Complex Selector

Use any CSS selector with querySelector

document.querySelector('.demo-item.special')
