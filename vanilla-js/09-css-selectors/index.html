Combinator Selectors

Select elements based on their relationship

CSS 
/* Descendant: any nested span */
.parent span {
  color: #22d3ee;
}

/* Child: direct children only */
.parent > p {
  border-left: 3px solid #a78bfa;
}

/* Adjacent Sibling: immediately after */
h4 + p {
  font-weight: bold;
}

üí° Key Point: Combinators: space (descendant), > (child), + (adjacent), ~ (general sibling)

Attribute Selectors

Select elements based on their attributes

CSS 
/* Has attribute */
[disabled] {
  opacity: 0.5;
  cursor: not-allowed;
}

/* Exact value */
input[type="email"] {
  border-color: #22d3ee;
}

/* Starts with */
a[href^="https"] {
  color: #34d399;
}

/* Ends with */
a[href$=".pdf"] {
  color: #ef4444;
}
Preview
HTTPS Link (green) HTTP Link (normal) Download PDF (red)
üí° Key Point: Attribute selectors: [attr], [attr=value], [attr^=start], [attr$=end], [attr*=contains]
