keyframes.css 
/* Define the animation */
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.1); }
}

/* Apply the animation */
.fade-in {
  animation: fadeIn 0.5s ease forwards;
}

.pulsing {
  animation: pulse 2s ease-in-out infinite;
}

‚ö° Performance Tips

‚úÖ DO Animate

  • transform (translate, scale, rotate)
  • opacity

These use GPU acceleration for smooth 60fps

‚ùå AVOID Animating

  • width, height
  • margin, padding
  • top, left, etc.

These trigger expensive layout recalculations

performance.css 
/* ‚ùå BAD - Animating width */
.element {
  width: 100px;
  transition: width 0.3s;
}
.element:hover {
  width: 200px;
}

/* ‚úÖ GOOD - Using transform instead */
.element {
  transform: scaleX(1);
  transition: transform 0.3s;
}
.element:hover {
  transform: scaleX(2);
}

/* Enable hardware acceleration */
.smooth-animation {
  will-change: transform, opacity;
}
